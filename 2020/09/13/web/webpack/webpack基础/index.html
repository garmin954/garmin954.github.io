webpack介绍啊

创建webpack项目

  1. 运行npm init -y 初始化项目
  2. 创建src源代码目录和dist产品目录
  3. 在src下创建index.html文件
  4. 安装webpack npm install webpack -d -g
  5. 创建 webpack.config.js 文件用于配置

webpack 配置

mode 当前项目环境

选项 描述
development 测试环境，会将 process.env.NODE_ENV 的值设为 development。
production 生产环境，会将 process.env.NODE_ENV 的值设为 production。

系统上线的时候使用该参数production，可以将生产环境和测试环境的配置区分，比如接口的域名
，测试和生产环境不是同一个域名
